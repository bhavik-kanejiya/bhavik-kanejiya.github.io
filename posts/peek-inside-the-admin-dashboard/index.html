Posted Jan 1, 2022
By Bhavik Kanejiya
1 min read

Application is vulnerable to broken access control allows employee to become the admin of a workspace.

Let’s do peek inside admin dashboard: Abuse API endpoint Header

Application introduction

It is a productivity app used by companies to track employee time and expenses.

Key Features

There are many features, but these are some of the important ones:

  • User can join many workspaces
  • User can create their own workspace
  • The admin of a workspace can invite other user by email

How does application works?

A user can join many workspaces from one account, but the user ID will be the same across all the workspaces. So, the user can create personal workspace along with the company’s workspace.
