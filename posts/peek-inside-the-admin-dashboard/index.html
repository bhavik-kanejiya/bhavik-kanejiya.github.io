Let’s do a peek inside the admin dashboard - Abuse API endpoint

Posted Jan 1, 2022
By Bhavik Kanejiya
1 min read

There is broken access control in the API which helps an employee to become the admin of a workspace but with limited permissions.

Let’s do peek inside admin dashboard: Abuse API endpoint Header

Application introduction

It is a productivity application with over two million users and is used by companies to track employee time and expenses.

Key Features

There are many features, but these are some of the important ones:

  • User can join multiple workspaces
  • User can create their own workspace
  • The admin of a workspace can invite other user by email

How does application works?

A user can join multiple workspaces from one account, but the user ID will be the same across all the workspaces. So, the user creates a personal workspace along with the company’s workspace.

The API uses the workspace ID for interaction with the application.
