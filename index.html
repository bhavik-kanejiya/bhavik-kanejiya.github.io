HTB Boardlight Walkthrough

Boardlight is a medium-difficulty Linux box on Hack The Box that showcases real-world privilege escalation via a vulnerable SUID binary. The machine walks you through basic enumeration, exploiting a web-based login system, and ultimately leveraging the CVE-2022-37706 vulnerability in Enlightenments enlightenment_sys utility to gain root access. Its an excellent box for anyone learning local privilege escalation techniques and digging into command injection through poorly sanitised SUID binaries.
